# psti-bot-telegram

Bot Telegram PSTI dengan Python Telegram Bot

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install pipenv :

```bash
pip3 install pipenv
pipenv install --system --dev
```

## Run
add your token to this script ```updater = ext.Updater('<TOKEN>')```
And run ```main.py``` :
```bash
python3 main.py
```
